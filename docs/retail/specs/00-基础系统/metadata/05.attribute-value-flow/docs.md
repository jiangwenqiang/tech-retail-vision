# 属性值维护流程

## 存储模型

属性值写入实体对应的属性值表：`*_attribute_value`。

## 维护流程

1. 挂载属性：创建 `attribute_id` 与记录ID关联行
2. 写入/修改值：对对应 value 列执行 upsert
3. 移除属性：删除该属性值关联行
4. 复合属性：挂父项时可联动挂子项

## 父项与子项的值

1. 子项承载业务值
2. 父复合属性通常仅作为结构标记
3. 父项记录通常存在，但其 value 列一般不承载业务值

## 约束建议

1. 删除子项前校验父项关系
2. 批量移除时优先处理复合父项
3. 所有变更写审计日志
