# 布局（Layout）

布局（Layout）用于定义实体在列表、详情、关系、侧边栏等视图中的字段与面板编排。

## 核心能力

1. 视图布局定义：按 `list/detail/relationships/sidePanel` 区分布局模型
2. 字段编排：控制字段顺序、分组、折叠、显隐
3. 关系视图编排：定义关系页签展示项与列结构
4. 布局分层：支持基线布局与租户覆盖布局

## 与 Client 的关系

1. `client` 负责组件行为与前端交互配置
2. `layout` 负责页面结构与字段编排配置
3. 两者共同决定最终 UI 呈现

## 建模建议

1. 布局配置与实体结构解耦，避免字段变更直接影响页面结构
2. 布局项引用字段时需做存在性校验
3. 发布前执行布局完整性校验（字段存在、关系可达、面板可渲染）
