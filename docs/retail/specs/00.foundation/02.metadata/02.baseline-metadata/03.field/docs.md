# 字段（Field）

字段（Field）是实体上的结构化数据单元，绑定具体字段类型并定义业务约束。

## 核心能力

1. 字段定义：名称、标签、类型、默认值
2. 约束控制：必填、唯一、可搜索、可排序
3. 数据治理：脱敏、审计、变更追踪
4. 显示协同：支持被客户端布局引用

## 关键关系

1. `field -> entity`（所属关系）
2. `field -> field_type`（参数契约关系）
3. `field -> attribute`（扩展关系）

## 与字段类型的契约边界

1. `field.type` 决定可用参数集合，参数定义来源于 `02.field-type` 的 `params`。
2. `field` 实例只应配置被 `field_type` 显式声明的参数，不应注入未声明参数。
3. 当 `field_type` 模板变更（参数增删、默认值调整）时，需要评估并回归所有引用该类型的字段实例。
