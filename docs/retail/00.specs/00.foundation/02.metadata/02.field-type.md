# Field Type Definition - 字段类型定义

## 作用

Field Type Definition 定义字段类型的契约，包括可接受的参数和验证规则。

## 核心字段类型（11种）

| 类型 | 代码 | 说明 | 参数白名单 |
|-----|------|------|-----------|
| 文本 | varchar | 可变长度文本 | maxLength, pattern, trim |
| 长文本 | text | 大段文本 | maxLength, rows |
| 整数 | int | 整数数值 | min, max |
| 小数 | decimal | 小数数值 | precision, scale |
| 布尔 | bool | 真/假值 | - |
| 日期 | date | 日期（不含时间） | - |
| 日期时间 | datetime | 日期时间 | - |
| 枚举 | enum | 单选枚举 | options |
| 多选 | multiEnum | 多选枚举 | options |
| 关联 | link | 关联单条记录 | foreignEntity, required |
| 多关联 | linkMultiple | 关联多条记录 | foreignEntity, required |

## 参数契约

### varchar 参数

```json
{
  "maxLength": 255,        // 最大长度（1-1000）
  "pattern": "^[a-zA-Z]+$", // 正则表达式
  "trim": true              // 保存前去首尾空格
}
```

### int 参数

```json
{
  "min": 0,    // 最小值
  "max": 100   // 最大值
}
```

### decimal 参数

```json
{
  "precision": 10,  // 总位数
  "scale": 2        // 小数位数
}
```

### enum 参数

```json
{
  "options": ["选项1", "选项2", "选项3"]
}
```

### link 参数

```json
{
  "foreignEntity": "Product",  // 关联实体
  "required": true             // 是否必填
}
```

## 基线与租户管理

### 平台基线字段类型

- 平台定义所有核心字段类型
- 租户可以直接使用
- 租户不能修改类型契约

### 租户使用

- 租户可以使用任何平台定义的字段类型
- 租户字段必须指定有效的字段类型
- 租户字段的 params 必须符合类型契约

## 验证规则

1. **类型白名单**：字段类型必须在预定义的 11 种类型中
2. **参数验证**：params 必须符合该类型的参数白名单
3. **参数类型**：参数值类型必须正确（如 maxLength 是整数）
4. **参数范围**：参数值必须在允许范围内
