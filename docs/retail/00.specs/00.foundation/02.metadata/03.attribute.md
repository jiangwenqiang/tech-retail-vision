# Attribute Definition - 属性扩展定义

## 作用

Attribute 定义实体的动态属性扩展，支持复合属性和多语言。

## 核心字段

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|-----|------|------|-------|------|
| code | VARCHAR(100) | 是 | - | 属性编码（实体内唯一） |
| name | VARCHAR(255) | 是 | - | 属性名称 |
| type | VARCHAR(50) | 是 | - | 属性类型（与字段类型相同） |
| scope | VARCHAR(20) | 是 | PLATFORM | 作用域（PLATFORM/TENANT） |
| customizable | BOOLEAN | 是 | true | 是否允许租户定制 |
| required | BOOLEAN | 是 | false | 是否必填 |
| isMultilang | BOOLEAN | 是 | false | 是否多语言 |
| parentId | BIGINT | 否 | - | 父属性ID（复合属性） |
| sortOrder | INT | 是 | - | 排序号 |
| yn | CHAR(1) | 是 | N | 软删除标记 |

## 复合属性

```json
{
  "code": "尺寸",
  "name": "尺寸",
  "type": "composite",
  "scope": "PLATFORM",
  "children": [
    {"code": "长度", "type": "decimal"},
    {"code": "宽度", "type": "decimal"},
    {"code": "高度", "type": "decimal"}
  ]
}
```

## 基线与租户管理

### 平台属性（PLATFORM）
- 平台定义，所有租户可见（除非通过 Customization 隐藏）
- 租户不能删除或修改类型
- 当 `customizable=true` 时，租户可通过 Customization 隐藏属性或覆盖属性

### 租户属性（TENANT）
- 租户自定义属性，仅当前租户可见
- 可以添加租户级别的 Attribute
- 支持复合属性结构

### 可见性控制

**租户级别**（通过 Customization）：
- 租户可以隐藏平台定义的属性（如果属性的 `customizable=true`）
